# Shelley Nails - Portfolio & IA Consultant

Site web professionnel pour Shelley Nails, artiste ongulaire, incluant une galerie, des tarifs détaillés et un consultant style par Intelligence Artificielle.

## Fonctionnalités

*   **Design Y2K / Edgy** : Interface soignée avec typographie gothique et serif.
*   **Designer IA** : Intégration de Gemini API pour générer des concepts de Nail Art sur mesure.
*   **Galerie** : Portfolio visuel des réalisations.
*   **Services & Tarifs** : Carte des prestations claire.
*   **Coaching** : Formulaire de demande de formation.

## Installation

1.  Cloner le projet
2.  Installer les dépendances : `npm install`
3.  Lancer le serveur : `npm run dev`

## Configuration API

Pour utiliser le Designer IA, une clé API Google Gemini est requise.
Le projet est configuré pour lire la clé depuis l'environnement ou via l'interface utilisateur.
